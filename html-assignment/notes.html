HTML Tags

Image Tag <img>
This tag helps you place an image in a website.
Footer <footer></footer>
This is a tag used to define a footer section.
Emphasis <em></em>
This tag is meant for readers to emphasize a slightly more important text. A few years ago, an italics tag was used instead of this one.
Ordered list Tag <ol></ol>
This is used for lists that have a specific order to them (For example, a list of the Star Wars movies in chronological order).
Form Tag <form></form>
This tag helps you create forms in a webpage (For example, a contact form).
